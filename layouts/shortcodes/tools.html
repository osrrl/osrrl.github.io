R is the statistical programming language we will learn about in this class. You can download R here. We will interface with R using RStudio, a fully feautred IDE. RStudio is available to download here.
Github is a Web-based Git version control repository hosting service. It is mostly used for computer code (like Dropbox for nerds). We will use GitHub for project management and sharing reproducible reports.
Slack is a communication platform we'll use to discuss specific topics outside of class. It also serves as a resource for students to ask questions and exchange information relevant to the course. You will receive an email with an invitation link to join the course Slack. For personal matters (only) you can email the professor.
